🔮 VISION PRODUIT
Ce que l'app fait
CourtVision AI transforme n'importe quel smartphone en analyste personnel de basket niveau professionnel. L'utilisateur filme son match ou son entraînement. L'IA analyse tout automatiquement et retourne un rapport complet, des conseils personnalisés, et un highlight reel cinématique.
Les 5 fonctionnalités révolutionnaires
1. Psychologie du joueur — Mental AI
Détection de la confiance via la gestuelle (posture, démarche, position des épaules)
Mental Fragility Score : l'IA détecte si tu joues sous pression ou détendu
Avant chaque match, message personnalisé : 'Ce soir, joue simple. Ton niveau de confiance est bas.'
Analyse du langage corporel après chaque tir raté
2. Reconstruction 3D depuis un seul téléphone
L'app reconstruit l'espace 3D complet du terrain depuis une seule caméra
Vue aérienne générée automatiquement : espaces, angles, trajectoires
Carte de chaleur 3D de toutes tes positions sur le terrain
Simulation des trajectoires de passes optimales
3. Jumeau Numérique — Digital Twin
L'app crée un avatar IA complet de toi basé sur tous tes matchs
Tu peux simuler des situations contre ta propre version améliorée
Les coachs testent des tactiques avec ton jumeau sans que tu sois présent
Comparaison automatique de ta technique vs les pros (pose estimation)
4. Scout IA — Analyse des adversaires
Tu scannes les anciens matchs de l'équipe adverse
L'IA génère un plan de jeu personnalisé contre cette équipe précise
Failles tactiques identifiées pour chaque adversaire
Mode recrutement : fiche joueur pro générée automatiquement
5. Production Cinématique Automatique
Highlight reel niveau ESPN généré en 2 minutes après le match
Commentaire IA dynamique sur tes meilleures actions
Musique adaptée automatiquement à l'intensité des actions
Partage direct sur Instagram, TikTok, YouTube

🤖 PIPELINE IA — ÉTAPE PAR ÉTAPE
Ce pipeline se déclenche après chaque upload vidéo. Il tourne en background via BullMQ. Construis chaque étape dans l'ordre exact.
Étape 1 — Prétraitement vidéo
Fichier : packages/ai/src/preprocessing.ts
Extraire les frames à 30fps avec FFmpeg
Détecter automatiquement les dimensions du terrain
Normaliser la perspective (correction fisheye si nécessaire)
Séparer les segments actifs vs inactifs (temps morts, coulisses)
Étape 2 — Détection et Tracking
Fichier : packages/ai/src/tracking.ts
MediaPipe Pose : détecter les 33 landmarks corporels sur chaque frame
ByteTrack : assigner un ID unique à chaque joueur et le tracker
Détecter le ballon avec un modèle YOLOv8 custom fine-tuné sur basket
Identifier le numéro de maillot de l'utilisateur principal
Étape 3 — Reconstruction 3D
Fichier : packages/ai/src/reconstruction3d.ts
DepthAnything v2 : estimer la profondeur depuis la caméra monoculaire
Mapper les coordonnées 2D vers coordonnées 3D terrain normalisé
Générer la vue aérienne du terrain avec positions en temps réel
Calculer les distances réelles entre joueurs
Étape 4 — Analyse des tirs
Fichier : packages/ai/src/shotAnalysis.ts
Détecter chaque tentative de tir (mouvement bras + trajectoire ballon)
Classifier : panier réussi, raté, contré, faute
Assigner la zone du terrain : corner 3pts, mi-distance, raquette, etc.
Analyser la posture de tir : angle coude, hauteur release, arc du ballon
Comparer la posture avec une base de données de tirs pro
Étape 5 — Analyse Psychologique
Fichier : packages/ai/src/mentalAnalysis.ts
Calculer le Mental Fragility Score basé sur les changements de posture
Détecter les patterns : baisse de tête après erreur, épaules tombantes, etc.
Analyser la vitesse de déplacement comme indicateur d'intensité
Comparer comportement début vs fin de match (fatigue mentale)
Étape 6 — Génération du Rapport IA
Fichier : packages/ai/src/reportGenerator.ts
Compiler toutes les données des étapes 1-5 en JSON structuré
Envoyer à Groq (Llama 3.3 70B) avec un prompt spécialisé coaching basket
Fallback automatique vers Ollama local si Groq est indisponible
Générer un rapport de 500-800 mots comme un vrai coach
Inclure : points forts, points faibles, exercices recommandés, objectifs
Étape 7 — Montage Highlight Reel
Fichier : packages/ai/src/highlightEditor.ts
Scorer chaque action : tir marqué = 10pts, passe décisive = 8pts, etc.
Sélectionner les 10-15 meilleures actions du match
FFmpeg : couper, assembler, ajouter transitions cinématiques
Coqui TTS : générer commentaire vocal IA sur les meilleures actions (open source, gratuit)
Ajouter overlay stats : nom, score, statistiques clés
Exporter en 1080p optimisé pour les réseaux sociaux

🆓 CONFIGURATION IA GRATUITE — GROQ + OLLAMA
Cette app est 100% gratuite grâce à Groq et Ollama. Voici comment les configurer correctement.
Groq — LLM principal (ultra rapide)
Groq est gratuit. Crée un compte sur groq.com, génère une clé API, c'est tout.
Modèle recommandé : llama-3.3-70b-versatile (meilleur rapport qualité/vitesse)
Limite gratuite : 14 400 requêtes/jour — largement suffisant pour débuter
Latence : ~0.5 seconde par rapport — beaucoup plus rapide que GPT-4o
Code d'intégration Groq — packages/ai/src/llm/groq.ts
import Groq from 'groq-sdk'
const groq = new Groq({ apiKey: process.env.GROQ_API_KEY })
export async function generateReport(analysisData: object): Promise<string> {
  const completion = await groq.chat.completions.create({
    model: 'llama-3.3-70b-versatile',
    messages: [{
      role: 'system',
      content: 'Tu es un coach basket professionnel niveau NBA.'
    }, {
      role: 'user',
      content: `Analyse ces données et génère un rapport: ${JSON.stringify(analysisData)}`
    }],
    max_tokens: 1000
  })
  return completion.choices[0].message.content ?? ''
}

Code d'intégration Ollama — packages/ai/src/llm/ollama.ts
export async function generateReportLocal(analysisData: object): Promise<string> {
  const response = await fetch('http://localhost:11434/api/generate', {
    method: 'POST',
    body: JSON.stringify({
      model: 'llama3.2',
      prompt: `Coach basket: ${JSON.stringify(analysisData)}`,
      stream: false
    })
  })
  const data = await response.json()
  return data.response
}
Système de fallback automatique
Crée un fichier packages/ai/src/llm/index.ts qui gère automatiquement les deux :
export async function generateReport(data: object): Promise<string> {
  try {
    return await generateReportGroq(data)   // Essaie Groq d'abord
  } catch {
    return await generateReportLocal(data)  // Fallback Ollama
  }
}
