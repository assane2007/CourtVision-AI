🤖 PIPELINE IA — 7 ÉTAPES (BullMQ)

Ce pipeline se déclenche après chaque upload vidéo. Il tourne en arrière-plan via BullMQ. Construis chaque étape dans l'ordre exact. Chaque étape émet un événement SSE pour la progression en temps réel.

Étape 1 — Prétraitement vidéo
Fichier : packages/ai/src/preprocessing.ts
Extraire les frames à 30fps avec FFmpeg
Détecter automatiquement les dimensions du terrain
Si calibration terrain disponible : mapper les 4 points pour la 3D
Normaliser la perspective (correction fisheye si nécessaire)
Séparer segments actifs vs inactifs (temps morts, coulisses)

Étape 2 — Détection et Tracking
Fichier : packages/ai/src/tracking.ts
MediaPipe Pose : détecter les 33 landmarks corporels sur chaque frame
ByteTrack : assigner un ID unique à chaque joueur et le tracker sur toute la durée
YOLOv8 custom fine-tuné : détecter le ballon frame par frame
Identifier le numéro de maillot de l'utilisateur principal

Étape 3 — Reconstruction 3D
Fichier : packages/ai/src/reconstruction3d.ts
DepthAnything v2 : estimer la profondeur depuis la caméra monoculaire
Si calibration disponible : mapper coordonnées 2D → 3D terrain normalisé
Générer la vue aérienne du terrain avec positions en temps réel
Calculer les distances réelles entre joueurs

Étape 4 — Analyse des tirs
Fichier : packages/ai/src/shotAnalysis.ts
Détecter chaque tentative de tir (mouvement bras + trajectoire ballon)
Classifier : panier réussi, raté, contré, faute
Assigner la zone terrain : corner 3pts, mi-distance, raquette, etc.
Analyser la posture de tir : angle coude, hauteur release, arc du ballon
Comparer la posture avec la base de données NBA open source

Étape 5 — Analyse Psychologique
Fichier : packages/ai/src/mentalAnalysis.ts
Calculer le Mental Fragility Score basé sur les changements de posture (sourced académiquement)
Détecter les patterns : baisse de tête après erreur, épaules tombantes, etc.
Analyser la vitesse de déplacement comme indicateur d'intensité et de fatigue
Comparer comportement début vs fin de match (fatigue mentale)
Si données HealthKit disponibles : croiser avec le HRV pour plus de précision

Étape 6 — Génération du Rapport + Programme
Fichier : packages/ai/src/reportGenerator.ts
Compiler toutes les données étapes 1-5 en JSON structuré
Envoyer à Groq (Llama 3.3 70B) avec prompt spécialisé coaching basket
Fallback automatique vers Ollama local si Groq indisponible
Rapport 500-800 mots : points forts, points faibles, exercices recommandés
Générer le programme d'entraînement des 7 prochains jours basé sur les faiblesses

Étape 7 — Montage Highlight Reel
Fichier : packages/ai/src/highlightEditor.ts
Scorer chaque action : tir marqué = 10pts, passe décisive = 8pts, etc.
Sélectionner les 10-15 meilleures actions du match
FFmpeg : couper, assembler, transitions cinématiques
Template au choix : cinéma / ESPN broadcast / hype TikTok
Coqui TTS : commentaire vocal IA sur les meilleures actions
Overlay stats : nom, score, statistiques clés + watermark CourtVision AI
Exporter en 1080p optimisé réseaux sociaux
